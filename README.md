# phone-duels
## What is it?
Phone Duels is a 2-player game. Each player battles it out in a sword fight using their phones as Joy-Con. 
## How will it work?
Each player will connect to the game via a QR code, which will be displayed on the laptop. Then the game will display the rules, and then get straight to the game.
